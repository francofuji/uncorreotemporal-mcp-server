## uncorreotemporal-mcp

Temporary email MCP server for autonomous agents that need real signup verification flows.

- Tools: `create_signup_inbox`, `wait_for_verification_email`, `get_latest_email`, `extract_otp_code`, `extract_verification_link`, `complete_signup_flow`
- Transports: `stdio` and `streamable-http`
- Install: `uvx uncorreotemporal-mcp`
- Repo: https://github.com/francofuji/uncorreotemporal-mcp-server
